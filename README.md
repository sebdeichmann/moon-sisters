# Moon Sisters Website

Responsive Rebuild der Moon-Sisters-Landingpage als Vite/React-Projekt.

## Entwicklung

```bash
npm install
npm run dev
```

Danach im Browser öffnen:

```text
http://127.0.0.1:5173
```

## Build

```bash
npm run build
```

Optional lokal ansehen:

```bash
npm run preview
```

## Struktur

- `src/content/siteContent.js` — zentrale Inhalte der Website
- `src/App.jsx` — Seitenstruktur und Komponenten
- `src/styles/global.css` — globales Styling und Responsive Design
- `public/assets/` — Bilder und generierte Assets
- `docs/` — Arbeitsberichte, Reviews und Responsive-Check-Screenshots
